# Quik

Go somewhere, quik.

quik lets you define bookmarked directories to quickly go to.
For example:

```bash
$ quik add "$HOME/Documents" docs

$ quik docs

```

## Quick Start

```bash
git clone https://github.com/mikeevmm/quik
cd quik
bash install.sh
quik --help
```
